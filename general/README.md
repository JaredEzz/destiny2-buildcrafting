# General — class-agnostic build-crafting (all 3 classes)

Everything in this folder applies to **Hunters, Titans, and Warlocks** alike. This is the shared base for building anything in the current sandbox.

## Start here

- **[`artifact-perks.md`](artifact-perks.md)** — ⭐ the big one. Every perk across all **7 Monument of Triumph artifacts**, grouped by **tier (T1/T2/T3)** with descriptions and ratings. The artifact is account-wide, so this is relevant no matter what class you play. Equip rule of thumb: the 7 equipped perks come from 5 columns (2 are Tier-1-only, 3 allow T1/T2, 2 allow T1/T2/T3) → you end up with **≥2 Tier-1 and ≤2 Tier-3** perks.
- **[`sandbox-changes.md`](sandbox-changes.md)** — what changed in the latest update: the reworked **6-stat system (0–200)**, armor 3.0/tiers, champion verbs, and the meta shifts that matter for building.
- **[`update-overview.md`](update-overview.md)** — high-level summary of the update.

## Supporting reference

- **[`subclasses/`](subclasses/)** — per-subclass mechanics (Arc, Solar, Void, Stasis, Strand, Prismatic). Written with Warlock aspect examples, but the **verbs, keywords, and fragments are universal** — scorch, ignition, suspend, unravel, sever, jolt, volatile, frost armor, devour, woven mail, transcendence, etc. all behave the same on every class. Read these for the mechanics; swap in your own class's aspects.
- **[`activities/`](activities/)** — where builds fit: activity index + where-to-play notes.
- **[`armor-farming.md`](armor-farming.md)** — how to farm good armor under the current stat system.

## Note for Hunter/Titan builders

The subclass docs name Warlock aspects (Voidwalker, Dawnblade, etc.), but the **fragment lists and elemental keyword behavior are shared across classes**. Use the artifact index + sandbox changes + the keyword mechanics directly; just substitute your class's aspects and exotic armor.
