# Destiny 2 Build-Crafting — Monument of Triumph (9.7.0)

A practical build-crafting reference for the **current Destiny 2 sandbox** (Monument of Triumph / Edge of Fate era, June 2026). Built Warlock-first, but the foundations here are **class-agnostic** — the artifact perks, the reworked stat system, champion verbs, and sandbox changes apply to **Hunters and Titans just the same**.

## How this repo is organized

| Folder | Who it's for | What's in it |
|---|---|---|
| **[`general/`](general/)** | **Everyone — all 3 classes** | The full tiered **artifact perk index**, what changed in the latest update, the new stat system, where-to-play activity notes, and subclass mechanics. Start here if you're building a Hunter or Titan. |
| **[`jared/`](jared/)** | Reference / examples | JaredEzz's personal **Warlock** build collection — worked examples of the ideas in `general/`. Warlock- and inventory-specific, so treat it as "here's how it's done," not a copy-paste for other classes. |

## The headline resource

**[`general/artifact-perks.md`](general/artifact-perks.md)** — every perk on all **7 Monument of Triumph artifacts**, sorted by tier (T1/T2/T3) with descriptions and ratings. This is the single most reusable thing here for any class, because the artifact is shared across your whole account.

## Quick caveats (read once)

- **Current as of June 2026 / sandbox v9.7.0.** Artifact perks rotate by season — this snapshot is the Monument of Triumph artifact set.
- **Artifact tiers** were reconstructed from datamined socket data and **cross-checked against in-game footage** (the live API manifest was frozen pre-update at the time of writing).
- **Not affiliated with Bungie.** Fan-made reference; verify anything load-bearing on your own character screen before committing to hard content.

## Contributing (Hunter/Titan friends 👀)

The `general/` material is the shared base. If you build out Hunter or Titan equivalents, add them in a sibling folder (e.g. `hunter/`, `titan/`) mirroring the `jared/` layout, and keep anything universal in `general/`.
