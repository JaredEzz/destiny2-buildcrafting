# Armor Farming — Getting Tier 5 Rolls for These Builds

> Gear tier scales with activity difficulty. **Tier 5 = guaranteed 30/25/20 stats, 11 mod energy, tuning slot.** Exotics are all retroactively Tier 5 — **only legendaries need farming.**

## Rule #1: the Ghost Armorer mod
Slot the **Armorer mod for your target archetype** in your Ghost's Economic socket **before farming** — it biases drops to that archetype at **1-in-2** (vs 1-in-3). Change it per build family.

## What to chase (archetype → builds)
| Ghost Armorer target | Archetype (30/25) | For these builds |
|---|---|---|
| **Colossus** | Super/Health | Skull Double Nova, Solipsism Nova, Well Lunafaction, Stormwrath |
| **Demolitionist** | Grenade/Class | Starfire, Getaway/Prismatic buddies, Dawnblade |
| **Siegebreaker** | Health/Grenade | Necrotic+Osteo tank, Threadweaver, Stasis turrets |
| **Reaver** | Class/Melee | Soul Siphon Voidwalker, Mataiodoxia, Hellion (dive spam) |
| **Bulwark** | Health/Class | Speaker's Sight, Boots of the Assembler, Nezarec's Sin |

Farm **one archetype set at a time** — the 19 builds cluster into these five.

## 🧍 Solo farms
1. **Distortions (new public events)** — base = T3, Heroic = T4, every drop can upgrade to **T5**. Zero-pressure; chain them with a poison/Hellion build.
2. **Expert → Master Lost Sectors** — daily solo rotation; Master pays high-tier gear + exotics. Match the day's champions.
3. **Portal activities (Solo Ops / Conquests)** at the highest tier you can clear — reward tier tracks difficulty, so one tier up beats farming a lower tier faster.
4. **Solo a featured dungeon** — the two weekly featured dungeons drop **farmable Tier 5**; pick the easy week (Spire, Warlord's Ruin) with Rime-Coat or Soul Siphon Voidwalker.

## 👥 Group farms
1. **Featured weekly raids + dungeons (2 each)** — farmable **Tier 5 + set bonuses**. This is where 2-piece/4-piece set hunting happens; check the set's bonus fits the build before committing the week.
2. **The Desert Perpetual** — after an encounter clear its **vendor sells highest-tier gear**: deterministic shopping for the exact missing slot/archetype. Best "last piece" source in the game.
3. **GM Nightfalls** — Tier 5 + adepts; set the Ghost mod and double-dip on trio nights.
4. **Pantheon 2.0 (Standard)** — weekly Tier 5 drops, no Feats required; Adventure difficulty (infinite revives) to learn bosses while still progressing.

## Three golden rules
1. **Ghost Armorer always on**, matched to the archetype you're chasing.
2. **Never re-farm exotics** — all Tier 5 with full tuning access.
3. **Masterwork + tune last** — masterworking boosts off-archetype stats and the T5 tuning slot moves 5 more points, so a "close" roll is fixable; only chase perfection on the two on-archetype stats.
