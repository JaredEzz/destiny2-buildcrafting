# The 4×5 Loadout Grid (20 Slots) — Real In-Game Icons & Colors

> Built from the **actual game assets** (21 icons, 22 color swatches, 22 preset names from the manifest). Tile captions read **`I# . C# . NAME`** — icon cycle position, color cycle position, preset name. All four options are **fully integrated 20-slot designs** — every build participates in the system; no bench rows.

**The reason for the organization:** each column is one **element wing, five deep, flowing as a color gradient** — so the grid reads both ways:
- **Pick by element:** go to your wing. **Night** (Void→Stasis, violet descending to navy), **Weave** (Strand greens blooming to rose), **Dawn** (Solar warming down to the gold healer), **Light** (slate brightening through silver/blue/lavender to magenta).
- **Pick by job:** the icon tells you — **I20 rays = GM · I16 arch = Boss · I11 sword = Adds · I12 shield = Team** — same glyphs in every wing.

## Setup workflow
1. Apply build from DIM → equip armor → set artifact perks in-game.
2. Save to the slot matching its grid position (row-major: slot 1 = top-left … slot 20 = bottom-right).
3. Cycle icon/color/name to the tile's code.
