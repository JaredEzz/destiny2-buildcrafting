# 🕷️ Spider-Man — Grapple-Uptime Variants (FOR FUN) — A/B test

Shared core: Strand Broodweaver · Grapple / Arcane Needle / Needlestorm / Healing Rift / Strafe Glide ·
Aspects Mindspun Invocation + The Wanderer · QSS+catalyst + Strand special + Strand heavy ·
Artifact Implement of Curiosity (Horde Shuttle, Refresh Threads, Elemental Benevolence, Threaded Blast,
Elemental Coalescence, Tangled Web, Elemental Overdrive).
Uptime levers (both): Thread of Generation · grapple-to-Tangle = free · mods Grenade Kickstart x2 + Bomber +
Momentum Transfer + Innervation · Healing Rift+Bomber · Refresh Threads (tangle->least-charged ability).

## Variant A — KARNSTEIN ARMLETS (sustain + strong uptime)
- Exotic: Karnstein Armlets (melee/finisher kill -> cure+restoration)
- Fragments: Generation, Fury, Warding, Continuity
- Stats: Grenade ~150 > Melee ~130 > Health ~100 > Class ~70
- Heal: vampiric melee. Forgiving everyday driver.

## Variant B — VERITY'S BROW (max grapple uptime)
- Exotic: Verity's Brow (helmet) — "The Fourth Magic": Strand WEAPON final blows -> Death Throes
  (grenade dmg bonus + grenade energy); throwing grapple with Death Throes -> allies gain grenade regen.
- Fragments: Generation, Fury, Warding, Continuity
- Stats: Grenade ~170 > Health ~110 > Class ~80 > Melee ~80 (Verity ignores Melee stat)
- Heal: Healing Rift + Woven Mail (Warding) + Recuperation/Better Already legs. Squishier.
- NOTE: QSS catalyst doubly required (makes QSS Strand so it procs Death Throes); lean on Strand special as main killer.

Net: Verity's = grapple returns much faster (constant weapon kills) but loses Karnstein's safety net.
