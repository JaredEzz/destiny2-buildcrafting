## 🧊 The 5 changes that reshape every build
1. Ability-cooldown bonus from stats cut **190%→125%** — uptime comes from kills/orbs/mods, not stat-stacking.
2. Super from boss damage cut **60%** — super leans on orbs + Super stat + self-refund weapons.
3. **Anti-Champion 2.0** — stun is intrinsic to weapon *frames*; Volatile/Radiant no longer stun Barrier.
4. All Exotic armor is **Tier 5** now (free tuning) — Exotic-defined builds got strictly stronger.
5. **Over-100 stat secondaries** are the new damage lever (Grenade +65% / Super +45% / Melee +30%).
