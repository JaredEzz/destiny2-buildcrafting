# Jared — personal Warlock build collection (reference examples)

JaredEzz's **Warlock** builds for the Monument of Triumph sandbox. This is **personal and inventory-specific** — it's the worked-example layer for the ideas in [`../general/`](../general/). Other classes won't copy-paste these (Warlock aspects/exotics), but they're useful as a model for *how* to spec and document a build.

## What's here

- **[`build-collection.md`](build-collection.md)** — the full collection: 20+ builds with subclass, exotic, aspects/fragments, weapons, stats, artifact perks, and gameplay loops.
- **[`builds-ranking.md`](builds-ranking.md)** — the builds ranked by how good/fun they are.
- **[`loadout-grid.md`](loadout-grid.md)** + **[`grids/`](grids/)** — the "Five Jobs" loadout grid (GM / Boss / Turret / Swarm / Team) used to organize 20 in-game loadout slots.
- **[`teammate-builds.md`](teammate-builds.md)** — complementary builds for a fireteam.
- **[`builds/`](builds/)** — individual deep-dives:
  - `exotic-helpers/` — one doc per exotic-anchored build (Necrotic+Osteo, Starfire, Getaway Artist, Rime-Coat/Osmiomancy stasis turrets, Speaker's Sight, Cenotaph, etc.).
  - `spiderman-grapple-vampire.md` + `spiderman-variants-grenade-uptime.md` — a "for fun" Strand grapple-tangle-melee build (Karnstein / Verity's Brow variants).
  - `eunoia-hellion-scorch-artillery.md` — a Solar Hellion + Eunoia scorch/ignition target build.
- **[`dim-import.md`](dim-import.md)** — notes for importing these into DIM.
- **[`tools/`](tools/)** — personal DIM/Bungie automation scripts (generate DIM links, apply fashion/ornaments, enrich loadouts, build the grid). These need your **own** Bungie/DIM auth to run — they read tokens/keys from local files that are **not** in this repo.

> Personal note: this is one player's Warlock setup. Mechanics and reasoning transfer; exact gear and instances do not.
